# Portfolio
### Version 1.0.0

### Installation
```bash
# Clone repo
git clone https://github.com/kroolar/portfolio

# Open  directory
cd portfolio

# Install packages
yarn install

# Run application
yarn start

# Run storybook
yarn storybook
```

### Tech Stack
- Create React App
- SCSS
- Tailwind
- Material Icons
- Storybook
- Typescript

### Deployment
```bash
yarn build
yarn build-storybook
```
